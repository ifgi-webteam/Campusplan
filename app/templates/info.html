Campusplan Informationen

Web-App Version 2.0

Die Campusplan Web-App wird als Open Source Projekt vom Institut für Geoinformatik gepflegt. Der gesamte Code für dieses Projekt befindet sich auf GitHub – Verbesserungsvorschläge von programmierfreudigen Entwicklern per Pull Request werden gern angenommen.

Bei Fragen, Wünschen und Fehlern schickt uns bitte eine Nachricht.

Feedback senden Github Repository

Copyright Hinweise

Die Kartendaten stammen von OpenStreetMap, die Kacheln von Mapquest Open.

Die App basiert zum größten Teil auf offenen Daten, die im Rahmen der Linked Open Data University of Münster Initiative auf data.uni-muenster.de zur Verfügung gestellt werden.

Die Daten der Mensen stammen von den Seiten des Studentenwerks Münster und stehen über eine inoffizielle API zur Nutzung bereit.

Die hier verwendeten Icons und Symbole (in der Navigation, auf den Buttons usw.) stammen teilweise von und/oder sind abgewandelt von Iconmonstr (Creative Commons Lizenz) und The Noun Project (Public Domain). Die Wettericons kommen von Lukas Bischoff & Erik Flowers (SIL OFL 1.1 & MIT License).
